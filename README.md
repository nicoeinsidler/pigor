# PIGOR

Pigor is a lightweight analysis tool for the NEPTUN beam line at Atominstitut of TU Wien, Austria. For more information visit [our homepage](http://www.neutroninterferometry.com/).

## How to install

To 'install' PIGOR, just install Python on your machine and run `python3 pigor.py` in the same directory as the pigor.py file is stored from your preferred bash/shell.
